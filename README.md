# Linux & Cloud Learning Journey

## Overview
This repository documents my hands-on learning path in Linux system administration and cloud fundamentals.
It is built while preparing for the LFCS (Linux Foundation Certified System Administrator) exam and for junior Linux/Cloud roles.

## Topics Covered
- Linux system administration fundamentals
- Users, groups, and permissions
- Bash scripting basics
- System services and process management
- Cloud fundamentals (AWS concepts)

## Repository Structure
- `linux/` – Linux administration notes and exercises
- `bash/` – Bash scripts and automation examples
- `cloud/` – Cloud-related notes and architectures
- `notes/` – Daily learning logs and summaries

## Goals
- Build strong Linux fundamentals
- Develop practical system administration skills
- Create a public, transparent learning record
